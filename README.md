# 🏠 Boston House Price Prediction

A Full Stack Machine Learning Web Application that predicts house prices based on housing features using a trained Machine Learning model.

🔗 **Live Demo:**
https://boston-house-price-prediction-3-f7m3.onrender.com/

📂 **GitHub Repository:**
https://github.com/khyathi15hue/boston-house-price-prediction

---

# 📌 Project Overview

This project predicts **house prices** using the Boston Housing Dataset.

Users enter housing features into a web interface, and the model predicts the estimated house price instantly.

This project demonstrates:

* Machine Learning Model Development
* Backend API using Flask
* Frontend UI using React
* Deployment using Render
* GitHub Version Control

---

# 🧠 Machine Learning Details

* Dataset: Boston Housing Dataset
* Algorithm Used: Linear Regression
* Libraries Used:

  * NumPy
  * Pandas
  * Scikit-learn
  * Pickle

The model predicts house prices using **13 input features**.

# 📊 Input Features

The model uses the following features:

1. CRIM – Crime Rate
2. ZN – Residential Land Zoned
3. INDUS – Non-retail Business Acres
4. CHAS – Charles River Dummy Variable
5. NOX – Nitric Oxide Concentration
6. RM – Average Rooms
7. AGE – Owner-Occupied Units Age
8. DIS – Distance to Employment Centers
9. RAD – Accessibility to Highways
10. TAX – Property Tax Rate
11. PTRATIO – Pupil-Teacher Ratio
12. B – Proportion of Population
13. LSTAT – Lower Status Population %

# 🚀 How to Run Locally

## Clone Repository

```bash
git clone https://github.com/khyathi15hue/boston-house-price-prediction.git
cd boston-house-price-prediction
```

---

## Install Backend Dependencies

```bash
pip install -r requirements.txt
```

---

## Run Backend

```bash
python app.py
```

---

## Run Frontend

```bash
cd frontend
npm install
npm start
```
